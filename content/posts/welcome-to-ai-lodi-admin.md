## What You Can Do

### Create Posts
- Simple password authentication (no OAuth needed)
- Beautiful form interface
- Full markdown support
- SEO optimization built-in

### Manage Content
- View all posts in a table
- Edit existing posts
- Delete posts easily
- Copy post links
- See live preview

### Auto-Deploy
- Every post you create is committed to GitHub
- Cloudflare Pages webhook triggers build
- All pages pre-generated as static HTML
- Lightning-fast CDN delivery

## Getting Started

1. Go to `/auth` and log in with your password
2. Click "Create New Post" in the dashboard
3. Fill in the form with your content
4. Choose draft or published status
5. Save - it commits to GitHub automatically!
